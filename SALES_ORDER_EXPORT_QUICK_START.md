# Sales Orders Export - Quick Start

## In 3 Steps

### 1. Open Sales Orders & Set Filters
```
Sidebar → Sales → Sales Orders
```
- Enter search term (optional)
- Select Payment Type (optional)
- Set Date From and Date To (optional)
- Filters update automatically

### 2. Click Export Button
- Red "Export" button in top right
- You'll see the preview page

### 3. Download PDF
Two formats available:
- **Summary Report** - Quick table view
- **Detailed Report** - Full order details

## Common Tasks

### Export All Orders for December
1. Set "Date From": 12/01/2024
2. Set "Date To": 12/31/2024
3. Click Export
4. Choose format
5. Download

### Export Cash Sales Only
1. Set "Payment Type": Cash
2. (Optional) Set date range
3. Click Export
4. Choose format
5. Download

### Export Customer Orders
1. Enter customer name in search
2. Click Export
3. Choose format
4. Download

### Print Orders Summary
1. Set filters as needed
2. Click Export
3. Click "Summary Report PDF"
4. Right-click → Print or Ctrl+P
5. Set orientation to Landscape (recommended)

## What Gets Exported

### Summary Report Includes
- SO Number
- Customer Name
- Item Count
- Amount (₱)
- Discount (₱)
- Paid (₱)
- Balance (₱)
- Payment Type
- Date

### Detailed Report Includes
**Per Order:**
- Customer details (name, phone, address)
- All items with prices
- Item breakdown (quantity, price, board feet)
- Order totals and discounts
- Payment information
- Order notes
- Order date/time

## Tips

- **Date Range**: Narrow dates = smaller PDFs
- **Search**: Partial matching works (e.g., "John" finds "John Doe")
- **Format**: Use Summary for lists, Detailed for records
- **Combine Filters**: Use multiple filters together
- **Preview First**: Always check preview before download
